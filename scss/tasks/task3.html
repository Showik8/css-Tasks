3. Parent Selector

The Ampersand (&)

& (Ampersand) მიუთითებს მშობელ სელექტორზე. ის გამოიყენება pseudo-class-ებისთვის (:hover) და BEM მოდიფიკატორებისთვის.

დავალება:

  1. შექმენი ღილაკის სტილი: 
    .btn {
  background: blue;
  &:hover { background: darkblue; }
  &--primary { font-weight: bold; }
}
