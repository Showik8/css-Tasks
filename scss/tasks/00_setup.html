00. Setup

Environment Configuration

სანამ კოდის წერას დავიწყებთ, საჭიროა გარემოს მომზადება. SCSS არის პრე-პროცესორი, რაც ნიშნავს რომ ის უნდა კომპილირდეს CSS-ში.

ნაბიჯი 1: Node.js & NPM

დარწმუნდი რომ გაქვს დაინსტალირებული Node.js.

node -v npm -v

ნაბიჯი 2: Sass Compiler

ჩვენ გამოვიყენებთ ოფიციალურ Dart Sass კომპილერს. გახსენი ტერმინალი და ჩაწერე:

npm install -g sass

ნაბიჯი 3: პროექტის შექმნა

  1. შექმენი საქაღალდე scss-course.
  2. გახსენი VS Code-ში.
  3. შექმენი ფაილი style.scss.
Watch Mode: ავტომატური კომპილაციისთვის გაუშვი:
sass --watch style.scss style.css
