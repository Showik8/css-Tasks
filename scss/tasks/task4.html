4. Partials

File Organization

Partials არის ფაილები, რომლებიც იწყება _ (underscore)-ით. ისინი არ კომპილირდება ცალკე CSS ფაილად, არამედ გამოიყენება სხვა ფაილებში.

დავალება:

  1. შექმენი _variables.scss და გადაიტანე იქ ფერები.
  2. შექმენი main.scss.
  3. გამოიყენე: @use 'variables';
Note: @import მოძველებულია. გამოიყენე @use.
