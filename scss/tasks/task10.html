#{} Syntax

Interpolation #{} გაძლევს საშუალებას გამოიყენო ცვლადი სელექტორის ან property-ს სახელში (და არა მარტო მნიშვნელობაში).

დავალება:

  1. შექმენი ცვლადი $name: 'danger';.
  2. გამოიყენე ინტერპოლაცია კლასის შესაქმნელად: 
    .btn-#{$name} {
  background: red;
}
