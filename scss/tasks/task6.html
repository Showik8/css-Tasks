6. Mixin Arguments

Dynamic Styles

Mixins-ს შეგვიძლია გადავცეთ არგუმენტები, როგორც ფუნქციებს.

დავალება:

  1. შექმენი @mixin button($bg, $color).
  2. გამოიყენე ის ორჯერ სხვადასხვა ფერებით: 
    .btn-primary {
  @include button(blue, white);
}
.btn-danger {
  @include button(red, white);
}
