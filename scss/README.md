## ðŸš€ Getting Started

### 1. Prerequisites
You need **Node.js** to install the Sass compiler.
- Check if installed: `node -v`
- [Download Node.js](https://nodejs.org/)

### 2. Install Sass
Open your terminal and run:
```bash
npm install -g sass
```

### 3. Usage (Local Development)
This course is interactive. For each task:
1.  Navigate to the `scss/tasks/` folder in your file explorer.
2.  Open the HTML file (e.g., `task1.html`) to see the instructions and target result.
3.  Create a `.scss` file (e.g., `task1.scss`) in your editor.
4.  Run the compiler in "Watch Mode" to automatically generate CSS:
    ```bash
    sass --watch task1.scss task1.css
    ```
5.  Link your generated CSS file in the HTML or inspect your results.
