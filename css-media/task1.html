თემა: Width & Height

აღწერა:

Media Queries ყველაზე ხშირად გამოიყენება ეკრანის სიგანისა და სიმაღლის მიხედვით სტილების შესაცვლელად.

  • width – viewport-ის ზუსტი სიგანე
  • min-width – მინიმალური სიგანე (ამ ზომიდან ზემოთ)
  • max-width – მაქსიმალური სიგანე (ამ ზომამდე)
  • height / min-height / max-height – იგივე ლოგიკა სიმაღლისთვის

ინსტრუქცია:

  1. გახსენი style.css (ან დაწერე <style> ტეგში).
  2. იპოვე ელემენტი კლასით .responsive-box.
  3. დაწერე მედია ქუერი, რომელიც გააქტიურდება როდესაც ეკრანის სიგანე ნაკლებია 800px-ზე:
    @media (max-width: 800px) { ... }
  4. ამ მედია ქუერიში შეუცვალე .responsive-box-ს ფერი (მაგ: background-color: #ef4444;).
  5. შეავიწროვე ბრაუზერის ფანჯარა და დააკვირდი ცვლილებას.
შენიშვნა: min-width ("Moble First") და max-width ("Desktop First") მიდგომები ყველაზე გავრცელებულია.
