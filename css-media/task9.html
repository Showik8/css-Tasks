თემა: Dark Mode

აღწერა:

თანამედროვე OS-ებს (macOS, Windows, iOS, Android) აქვთ სისტემური Dark Mode. ჩვენ შეგვიძლია ამას მოვუსმინოთ.

@media (prefers-color-scheme: dark) { ... }

ინსტრუქცია:

  1. გახსენი style.css.
  2. იპოვე .theme-card.
  3. დაწერე მედია ქუერი prefers-color-scheme: dark.
  4. Dark Mode-ში:
    • .theme-card-ის ფონი გახადე შავი (#1e293b).
    • ტექსტის ფერი გახადე თეთრი.
  5. ტესტირება: Chrome DevTools -> Rendering -> Emulate CSS media feature prefers-color-scheme: dark (ან შეცვალე სისტემური პარამეტრები).
რჩევა: საუკეთესო პრაქტიკაა CSS ცვლადების (Custom Properties) გამოყენება და მათი შეცვლა :root ბლოკში მედია ქუერის შიგნით.
