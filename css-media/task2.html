თემა: Device & Orientation

აღწერა:

ეს მახასიათებლები ეხება ფიზიკურ მოწყობილობას და არა მხოლოდ ბრაუზერის ფანჯარას.

  • device-width/height – ეკრანის ფიზიკური რეზოლუცია
  • orientation – portrait (ვერტიკალური) ან landscape (ჰორიზონტალური)

ინსტრუქცია:

  1. გახსენი style.css.
  2. იპოვე ელემენტი .device-box.
  3. დაწერე მედია ქუერი ორიენტაციისთვის:
    @media (orientation: landscape) { ... }
  4. Landscape რეჟიმში შეცვალე .device-box-ის სიგანე width: 500px;-ზე და ფერი სხვა ფერზე.
  5. შენიშვნა: დესკტოპზე ეს მარტივია – თუ ფანჯრის სიგანე მეტია სიმაღლეზე, ეს ითვლება landscape-ად.
რჩევა: device-width ხშირად გამოიყენებოდა ძველ მობილურ საიტებში, თუმცა დღესდღეობით უმჯობესია ჩვეულებრივი width-ის გამოყენება.
