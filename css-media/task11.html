პროექტი: Responsive Navigation

აღწერა:

ეს არის რეალური პროექტის სიმულაცია. გვაქვს ნავიგაციის პანელი (Header).

დესკტოპზე ბმულები ჩანს, მაგრამ მობილურზე ისინი უნდა დაიმალოს და გამოჩნდეს "Hamburger" მენიუ.

ინსტრუქცია:

  1. გახსენი style.css და იპოვე .nav-demo სექცია.
  2. დაწერე მედია ქუერი 768px-ის ქვემოთ (max-width: 768px).
  3. ამ ზომაზე:
    • დამალე .nav-links (display: none).
    • გამოაჩინე .hamburger (display: block).
  4. ბონუსი: სცადე .nav-links-ის ისე შეცვლა, რომ არ გაქრეს, არამედ ჩამოვიდეს ახალ ხაზზე (flex-direction: column) და დაიკავოს 100% სიგანე.
შენიშვნა: რეალურ პროექტებში Hamburger-ზე დაჭერისას JavaScript-ით ვამატებთ კლასს (მაგ: .open), რომელიც აჩენს მენიუს. აქ მხოლოდ CSS Layout-ზე ვმუშაობთ.
