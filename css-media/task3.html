თემა: Resolution & Display

აღწერა:

ეს პარამეტრები ეკრანის ხარისხს და ფერების შესაძლებლობებს განსაზღვრავს.

  • resolution – პიქსელების სიმკვრივე (DPI / dppx). მაგ: Retina ეკრანები.
  • aspect-ratio – ეკრანის პროპორცია (მაგ: 16/9).
  • monochrome – შავ-თეთრი ეკრანები.
  • color – ფერების ბიტები.

ინსტრუქცია:

  1. გახსენი style.css.
  2. იპოვე ელემენტი .resolution-box.
  3. დაწერე მედია ქუერი მაღალი გაფართოების ეკრანებისთვის (Retina):
    @media (min-resolution: 2dppx) { ... }
  4. შეცვალე .resolution-box-ის ბორდერი: border: 5px solid white;.
  5. (სურვილისამებრ) სცადე monochrome-ის ტესტირებაც (Chrome DevTools-ში Rendering ტაბიდან შეიძლება ემულაცია).
შენიშვნა: dppx ნიშნავს dots per pixel unit. 1dppx არის სტანდარტული, 2dppx არის Retina (High DPI).
