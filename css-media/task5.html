თემა: Other Features

აღწერა:

სხვა მნიშვნელოვანი მედია მახასიათებლები:

  • update – ეკრანის განახლების სიჩქარე (none, slow, fast). მაგ: ელექტრონული წიგნები (Kindle) არის slow.
  • grid – განსაზღვრავს მოწყობილობა ბადისებრია (tty) თუ ბიტმაპური (ეკრანი).

ინსტრუქცია:

  1. გახსენი style.css.
  2. იპოვე .feature-item რომელშიც სპინერი ტრიალებს.
  3. დაწერე კოდი, რომელიც შეაჩერებს ანიმაციას თუ ეკრანი არის slow (მაგ: e-ink):
    @media (update: slow) { .spinner { animation: none; } }
  4. დამატებით: შეგიძლია შეამოწმო @media (grid: 0) (რაც ნიშნავს რომ ჩვეულებრივი ბიტმაპ ეკრანია).
რჩევა: update: slow-ს დროს კარგი პრაქტიკაა ანიმაციების გამორთვა, რადგან e-ink ეკრანებზე ეს ცუდად ჩანს.
