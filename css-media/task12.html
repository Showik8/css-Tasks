პროექტი: E-commerce Grid

აღწერა:

ელ-კომერციის საიტებზე პროდუქტების ბადე (Grid) უნდა ადაპტირდებოდეს ეკრანის ზომასთან.

ამჟამად გვაქვს 3 სვეტი (grid-template-columns: repeat(3, 1fr)).

ინსტრუქცია (Mobile-First მიდგომა):

  1. გახსენი style.css და იპოვე .product-grid.
  2. შეცვალე .product-grid-ის საბაზისო სტილი ისე, რომ იყოს 1 სვეტი (Mobile).
  3. დაამატე მედია ქუერი 600px-ის ზევით:
    @media (min-width: 600px) { ... grid-template-columns: repeat(2, 1fr); }
  4. დაამატე მედია ქუერი 1000px-ის ზევით:
    @media (min-width: 1000px) { ... grid-template-columns: repeat(3, 1fr); }
რჩევა: თანამედროვე Grid-ში შეიძლება repeat(auto-fit, minmax(250px, 1fr)) გამოყენებაც, რაც მედია ქუერების გარეშეც მუშაობს, მაგრამ კარგია თუ ორივე მეთოდს ფლობთ.
