თემა: Reduced Motion

აღწერა:

ბევრ მომხმარებელს აწუხებს ვესტიბულური დარღვევები და სწრაფი ანიმაციები მათთვის დისკომფორტს იწვევს. მათ შეუძლიათ სისტემაში ჩართონ "Reduce Motion".

@media (prefers-reduced-motion: reduce) { ... }

ინსტრუქცია:

  1. გახსენი style.css.
  2. იპოვე .motion-box, რომელიც სწრაფად ბრუნავს.
  3. დაწერე მედია ქუერი prefers-reduced-motion: reduce.
  4. ამ შემთხვევაში ანიმაცია მთლიანად გათიშე: animation: none;.
  5. ტესტირება: Chrome DevTools -> Rendering -> Emulate CSS media feature prefers-reduced-motion: reduce.
რჩევა: რეალურ პროექტებში, ნაცვლად მთლიანად გათიშვისა, ხანდახან ვანელებთ ანიმაციას ან ვიყენებთ მარტივ "fade" ეფექტს მოძრაობის ნაცვლად.
