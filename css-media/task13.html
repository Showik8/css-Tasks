პროექტი: Hero Section

აღწერა:

Hero სექცია არის საიტის პირველი ნაწილი. დესკტოპზე ხშირად გვაქვს ტექსტი მარცხნივ და სურათი მარჯვნივ (2 სვეტი).

მობილურზე ეს უნდა შეიცვალოს: სურათი ავიდეს ზემოთ, ტექსტი ჩამოვიდეს ქვემოთ (1 სვეტი).

ინსტრუქცია:

  1. გახსენი style.css და იპოვე .hero-section.
  2. ამჟამად ის არის grid-template-columns: 1fr 1fr (2 სვეტი).
  3. დაწერე მედია ქუერი 800px-ის ქვემოთ (max-width: 800px).
  4. ამ ზომაზე:
    • შეცვალე Grid 1 სვეტად (grid-template-columns: 1fr).
    • შეცვალე .hero-image-ის სიმაღლე (მაგ: height: 200px).
    • დაამატე order: -1; სურათს, რომ ის ტექსტის ზემოთ გამოჩნდეს (grid-ში შეგვიძლია ელემენტების თანმიმდევრობის შეცვლა HTML-ის შეუცვლელად).
რჩევა: order ძალიან მძლავრი ინსტრუმენტია Flexbox და Grid-ში რესპონსიული დიზაინისთვის.
