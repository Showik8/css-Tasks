თემა: Print Media

აღწერა:

ვებ-გვერდების ბეჭდვის დროს (Ctrl+P / Cmd+P) ხშირად გვინდა ზედმეტი ელემენტების დამალვა (მაგ: ნავიგაცია, ღილაკები, ფერადი ფონები).

@media print { ... }

ინსტრუქცია:

  1. გახსენი style.css.
  2. დაწერე @media print ბლოკი.
  3. ამ ბლოკში:
    • დამალე მარცხენა მენიუ (.instructions-panel { display: none; }).
    • .print-card-ს მოაშორე ჩრდილი და ბორდერი.
    • გამოაჩინე .print-only კლასის მქონე ელემენტი (display: block).
  4. დააჭირე Cmd+P (Mac) ან Ctrl+P (Windows) და ნახე Preview.
რჩევა: ბეჭდვის დროს ფერადი ფონები ავტომატურად ითიშება ბრაუზერის მიერ, თუ მომხმარებელმა არ ჩართო "Background graphics".
