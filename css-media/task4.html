დავალება 4

თემა: Light & Interaction

აღწერა:

ეს მახასიათებლები გვაძლევს საშუალებას გავიგოთ, როგორ ურთიერთქმედებს მომხმარებელი საიტთან (მაუსით თუ თითით).

  • hover: hover – მოწყობილობას აქვს hover-ის მხარდაჭერა (მაგ: მაუსი).
  • hover: none – არ აქვს (მაგ: სმარტფონი).
  • pointer: fine – ზუსტი მაჩვენებელი (მაუსი).
  • pointer: coarse – უხეში მაჩვენებელი (თითი).

ინსტრუქცია:

  1. გახსენი style.css.
  2. იპოვე ღილაკი .interaction-btn.
  3. დაწერე კოდი, რომელიც გაზრდის ღილაკის ზომას (padding) თუ მომხმარებელი სენსორულ ეკრანზეა:
    @media (pointer: coarse) { ... }
  4. ასევე, შეგიძლია დაამატო ეფექტები მხოლოდ მაშინ, თუ hover მხარდაჭერილია:
    @media (hover: hover) { .interaction-btn:hover { ... } }
შენიშვნა: inverted-colors გამოიყენება ხელმისაწვდომობისთვის (Accessibility), როცა მომხმარებელს ფერების ინვერსია აქვს ჩართული.
