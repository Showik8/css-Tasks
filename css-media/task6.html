თემა: Breakpoint Ranges

აღწერა:

ხშირად გვჭირდება სტილები, რომლებიც მოქმედებს მხოლოდ გარკვეულ შუალედში (მაგალითად, მხოლოდ პლანშეტებზე - 600px-დან 900px-მდე).

ამისათვის ვიყენებთ and ოპერატორს:

@media (min-width: 600px) and (max-width: 900px) { ... }

ინსტრუქცია:

  1. გახსენი style.css (ან დაწერე <style>-ში).
  2. იპოვე ელემენტი .range-box.
  3. დაწერე მედია ქუერი, რომელიც გააქტიურდება 600px-სა და 900px-ს შორის.
  4. ამ შუალედში შეუცვალე .range-box-ს ფერი ყვითელზე (#f59e0b) და ტექსტი გახადე "Tablet Mode".
  5. დაამატე მეორე მედია ქუერი 900px-ის ზევით, სადაც ფერი გახდება მწვანე (#10b981) და ტექსტი "Desktop Mode".
შენიშვნა: ახალ CSS სინტაქსში შეგვიძლია დავწეროთ: @media (600px <= width <= 900px), რაც უფრო იკითხებადია.
