# CSS Mastery Tasks

Welcome to the CSS Mastery Tasks repository. This project contains a series of learning modules designed to help you master various aspects of CSS, from layout techniques to advanced styling.

## Modules

Below is a list of the available learning modules. Each folder contains a specific set of tasks. To start a module, navigate to the folder and open `index.html` in your browser.

-   **[CSS Flexbox](./css-flexbox)** - Master the Flexible Box Layout module. Learn to align and distribute space among items in a container.
-   **[CSS Grid](./css-grid)** - Dive into the CSS Grid Layout module. Create complex, responsive web design layouts with ease.
-   **[CSS Position](./css-position)** - Understand how to position elements using `relative`, `absolute`, `fixed`, and `sticky`.
-   **[CSS Pseudo-classes](./css-pseudo)** - Explore pseudo-classes to style elements based on their state (e.g., `:hover`, `:focus`, `:nth-child`).
-   **[CSS General Tasks](./css-task)** - A collection of general CSS challenges to test and refine your overall skills.

## How to Use

1.  Open the folder of the module you wish to practice.
2.  Open `index.html` to view the list of tasks for that module.
3.  Complete the tasks by editing the corresponding HTML and CSS files.
