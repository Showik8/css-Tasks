დავალება 13

opacity & scale (pulse)

აღწერა:

scale და opacity კომბინაციით შეგვიძლია მივიღოთ ძალიან ეფექტური "პულსაციის" ეფექტი.

ინსტრუქცია:

  1. გვინდა გული ცემდეს და რგოლები გამოდიოდეს.
  2. გული: შექმენი @keyframes heartbeat:
    0%, 100% { transform: scale(1); }
    50% { transform: scale(1.1); }
    მიანიჭე .heart-ს 0.8s infinite ease-in-out.
  3. რგოლი: შექმენი @keyframes ripple:
    from { transform: scale(1); opacity: 0.6; }
    to { transform: scale(3); opacity: 0; }
    მიანიჭე .pulse-ring-ს 1.5s infinite linear.
