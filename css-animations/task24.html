Accordion Reveal (Grid Transition)

აღწერა:

აკორდეონის (ჩამოსაშლელი ტექსტის) ანიმაცია `height: auto`-ზე რთულია, მაგრამ `grid` ამას აგვარებს.

ინსტრუქცია:

  1. გვაქვს აკორდეონი .accordion-content, რომელიც თავიდან დამალულია (height: 0 ვერსია).
  2. იპოვე .accordion-item:hover .accordion-content სელექტორი.
  3. შენი მიზანია შეცვალო grid-template-rows 0fr-დან 1fr-მდე.
  4. დააკვირდი როგორ "გამოჩოჩდება" კონტენტი ავტომატურად.

Why use CSS Grid for Animation? ▼
Because it allows us to transition from 0 to auto height using the fractional unit (fr) trick, which was impossible with height property alone!
