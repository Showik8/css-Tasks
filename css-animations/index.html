CSS ANIMATIONS

Visual Cheat Sheet

Animation Shorthand

ყველა პარამეტრის ერთ ხაზში ჩაწერის წესი:

animation: [name] [duration] [timing-function] [delay] [iteration-count] [direction] [fill-mode] [play-state];

/* მაგალითი: */
animation: bounce 2s ease-in-out 0.5s infinite alternate forwards running;

@keyframes

განსაზღვრავს ანიმაციის კადრებს.

From / To 
@keyframes slide {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}
Percentages (პროცენტები) 
@keyframes colorChange {
  0% { background: red; }
  50% { background: yellow; }
  100% { background: red; }
}

Properties Detail

animation-duration

რამდენ ხანს გრძელდება 1 ციკლი (s ან ms).

animation-duration: 2s;      /* 2 წამი */
animation-duration: 500ms;   /* 0.5 წამი */
animation-timing-function

სიჩქარის განაწილება დროში.

ease (default) linear (თანაბარი) ease-in (აჩქარება) ease-out (შენელება) steps(n) (ბიჯები) cubic-bezier(n,n,n,n)
animation-delay

დაყოვნება დაწყებამდე.

animation-delay: 1s; /* დაიწყება 1 წამის შემდეგ */
animation-iteration-count

რამდენჯერ განმეორდეს.

animation-iteration-count: 3;        /* 3-ჯერ */
animation-iteration-count: infinite; /* უსასრულოდ */
animation-direction

ანიმაციის მიმართულება.

animation-direction: normal;            /* წინ */
animation-direction: reverse;           /* უკან */
animation-direction: alternate;         /* წინ -> უკან */
animation-direction: alternate-reverse; /* უკან -> წინ */
animation-fill-mode

სტილები დაწყებამდე/დასრულების შემდეგ.

animation-fill-mode: forwards;  /* ინარჩუნებს ბოლო კადრს */
animation-fill-mode: backwards; /* იღებს პირველ კადრს delay-ს დროს */
animation-fill-mode: both;      /* ორივე */
