Flight Button (Transform)

აღწერა:

ანიმაციებში ყველაზე ხშირად ვიყენებთ transform პროპერთის, რადგან ის არ იწვევს გვერდის გადახატვას (repaint).

ინსტრუქცია:

  1. ეს არის "Flight Button". დაწკაპუნებისას თვითმფრინავი უნდა გაფრინდეს.
  2. იპოვე @keyframes flyAway style.css-ში.
  3. დაამატე:
    • 100%: transform: translate(200px, -100px) rotate(45deg); opacity: 0;
  4. შემდეგ მიანიჭე ის .flight-btn:active .plane-icon-ს:
    animation: flyAway 0.8s ease-in forwards;
  5. დააკვირდი რომ თვითმფრინავი ღილაკის საზღვრებს გარეთ გადის (overflow: hidden მოხსნილია).
