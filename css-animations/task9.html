Music Visualizer (Play State)

აღწერა:

animation-play-state განსაზღვრავს ანიმაცია გაშვებულია (running) თუ შეჩერებული (paused).

ინსტრუქცია:

  1. ეს არის მუსიკალური ვიზუალიზატორი (ექვალაიზერი).
  2. მუსიკა უკრავს. დავალებაა: გაჩერდეს ვიზუალიზაცია, როცა მომხმარებელი მაუსს მიიტანს (თითქოს "Pause" ღილაკს დააწვა).
  3. იპოვე ან დაამატე css-ში:
    .music-player:hover .bar { animation-play-state: paused; }
  4. სცადე მაუსის მიტანა და გაწევა.
