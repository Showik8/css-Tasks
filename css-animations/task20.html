Hero Section (Composition)

рЃљрЃдрЃгрЃћрЃарЃљ:

рЃњрЃўрЃџрЃЮрЃфрЃљрЃЋ! рЃерЃћрЃю рЃњрЃљрЃўрЃљрЃарЃћ CSS рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃў.

рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ:

  1. рЃћрЃА рЃљрЃарЃўрЃА рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃўрЃА "Hero Section".
  2. рЃљрЃЦ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ:
    • рЃерЃћрЃЏрЃЮрЃАрЃЋрЃџрЃљ (Fade Up) рЃбрЃћрЃЦрЃАрЃбрЃќрЃћ.
    • рЃЏрЃЮрЃџрЃўрЃЋрЃџрЃўрЃЋрЃћ (Floating) рЃцрЃўрЃњрЃБрЃарЃћрЃЉрЃў рЃцрЃЮрЃюрЃќрЃћ.
    • рЃЎрЃЮрЃЏрЃърЃЮрЃќрЃўрЃфрЃўрЃљ рЃЦрЃЏрЃюрЃўрЃА "рЃфрЃЮрЃфрЃ«рЃљрЃџ" рЃћрЃцрЃћрЃЦрЃбрЃА.
  3. рЃћрЃА рЃљрЃарЃўрЃА рЃарЃћрЃљрЃџрЃБрЃарЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃўрЃЏрЃўрЃАрЃљ, рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃћрЃЉрЃА рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃАрЃљрЃўрЃбрЃћрЃЉрЃќрЃћ рЃЋрЃўрЃќрЃўрЃбрЃЮрЃарЃўрЃА рЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃўрЃА рЃЏрЃўрЃАрЃљрЃърЃДрЃарЃЮрЃЉрЃљрЃЊ.
  4. рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃћрЃЉрЃў! ­ЪјЅ
