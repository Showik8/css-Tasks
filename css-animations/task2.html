animation-name & @keyframes

აღწერა:

animation-name განსაზღვრავს რომელი @keyframes ანიმაცია უნდა შესრულდეს. @keyframes აღწერს რა უნდა მოხდეს ანიმაციის დროს.

ინსტრუქცია:

  1. გახსენი style.css და იპოვე @keyframes slideRight.
  2. განსაზღვრე ანიმაცია:
    0% { transform: translateX(0); }
    100% { transform: translateX(400px); }
  3. შემდეგ იპოვე კლასი .task2-box და მიანიჭე:
    animation-name: slideRight;
    animation-duration: 3s;
    animation-fill-mode: forwards;
