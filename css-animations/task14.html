Dark Mode Switch (Color Transition)

აღწერა:

ანიმაცია შეგვიძლია გავუკეთოთ ფერებსაც (background, border-color, color).

ინსტრუქცია:

  1. აქ გვაქვს ავტომატური "Dark Mode" გადამრთველი.
  2. ნახე @keyframes bgChange და @keyframes toggleMove.
  3. შენი დავალებაა ჩართო ანიმაციები. მიანიჭე:
    .switch-container -> animation: bgChange 5s infinite alternate;
    .toggle-thumb -> animation: toggleMove 5s infinite alternate;
  4. დააკვირდი ფონისა და ღილაკის ფერის სინქრონულ ცვლილებას.
