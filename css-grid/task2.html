Task 2: Grid Placement

ელემენტების განლაგება: grid-column & grid-row

აღწერა

Grid-ში ელემენტები შეგვიძლია გავწელოთ ან გადავაადგილოთ "ხაზების" (lines) ნომრების მიხედვით.

ინსტრუქცია

  1. გვაქვს 4x4 ბადე (grid).
  2. იპოვე ელემენტი კლასით .header-item. გაწელე ის მთელ სიგანეზე (1-დან 5-მდე ხაზამდე):
    grid-column: 1 / 5;
  3. იპოვე ელემენტი კლასით .sidebar-item. განათავსე ის მარცხენა მხარეს, მე-2 რიგიდან ბოლომდე:
    grid-row: 2 / 5;
  4. იპოვე .main-item და გაწელე დარჩენილ სივრცეში:
    grid-column: 2 / 5;
    grid-row: 2 / 5;
შენიშვნა: ხაზების ნომრები იწყება 1-დან. 4 სვეტიან grid-ს აქვს 5 ხაზი (1, 2, 3, 4, 5).
