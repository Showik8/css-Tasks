Task 8: Overlap & Layering

Advanced: ფენები და გადაფარვა

აღწერა

Grid-ის ერთ-ერთი ყველაზე ძლიერი მხარეა ელემენტების გადაფარვა (Overlap). Flexbox-ისგან განსხვავებით, აქ შეგვიძლია რამდენიმე ელემენტი ერთსადაიმავე უჯრაში მოვათავსოთ.

ინსტრუქცია

  1. ჩვენ გვაქვს 3 ელემენტი: ბაზა (base), ბარათი (card) და დეკორაცია (decoration).
  2. განათავსე .t8-base მთლიან Grid-ზე:
    grid-column: 1 / -1; grid-row: 1 / -1;
  3. განათავსე .t8-card ცენტრში:
    grid-column: 2 / 5; grid-row: 2 / 4;
    მიანიჭე z-index: 10; რომ წინ წამოვიდეს.
  4. განათავსე .t8-decoration ისე რომ ნაწილობრივ ფარავდეს ბარათს:
    grid-column: 4 / 6; grid-row: 1 / 3;
    მიანიჭე z-index: 5; (ბარათის უკან, მაგრამ ბაზის წინ).
Broken Grid: ეს ტექნიკა გამოიყენება თანამედროვე "Broken Grid" დიზაინებში, სადაც ელემენტები არასტანდარტულად კვეთენ ერთმანეთს.
