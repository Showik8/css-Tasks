Task 1: Grid Basics

შესავალი: display: grid & templates

აღწერა

Grid არის ორგანზომილებიანი სისტემა. ჩვენ შეგვიძლია განვსაზღვროთ როგორც სვეტები (columns), ისე სტრიქონები (rows).

ინსტრუქცია

  1. იპოვე კონტეინერი კლასით .task1-container.
  2. შეცვალე display: block და გახადე display: grid.
  3. განსაზღვრე 3 სვეტი: grid-template-columns: 1fr 1fr 1fr; (ანუ სამი თანაბარი ნაწილი).
  4. დაამატე დაშორება: gap: 20px;.
შენიშვნა: fr (fraction) არის სპეციალური ერთეული grid-ისთვის, რომელიც იკავებს დარჩენილ თავისუფალ ადგილს.
