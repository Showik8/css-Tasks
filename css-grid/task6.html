Task 6: Functions

ფუნქციები: repeat() და minmax()

აღწერა

Grid-ის წერის გასამარტივებლად და უფრო მოქნილი დიზაინისთვის ვიყენებთ ფუნქციებს.

ინსტრუქცია

  1. ამჟამად გვაქვს 3 სვეტი: 1fr 1fr 1fr.
  2. გადაწერე ეს repeat() ფუნქციის გამოყენებით:
    grid-template-columns: repeat(3, 1fr);
  3. ახლა სცადე minmax(). გააკეთე ისე რომ სვეტები არასდროს იყოს 100px-ზე პატარა, მაგრამ გაიზარდოს დარჩენილ ადგილამდე:
    grid-template-columns: repeat(3, minmax(100px, 1fr));
  4. სცადე შეავიწროვო ბრაუზერის ფანჯარა და დააკვირდი როგორ იქცევიან სვეტები.
minmax(min, max) ძალიან სასარგებლოა მობილური დიზაინისთვის, რათა ელემენტები ძალიან არ დაპატარავდნენ.
