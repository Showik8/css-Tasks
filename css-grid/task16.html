Task 16: Mosaic Gallery

Dense packing & spans

აღწერა

შევქმნათ "Pinterest"-ის სტილის (Masonry მსგავსი) გალერეა. Grid ამას სრულყოფილად ვერ აკეთებს (JavaScript გარეშე სვეტებში სიმაღლეები ვერ იცვლება), მაგრამ dense და span გვაძლევს ძალიან ლამაზი მოზაიკის ეფექტს.

ინსტრუქცია

  1. გვაქვს ბევრი სურათი. ზოგს აქვს კლასი .h-2 (სიმაღლეში 2), ზოგს .w-2 (სიგანეში 2).
  2. ამჟამად ისინი ტოვებენ ცარიელ ადგილებს.
  3. დაამატე: grid-auto-flow: dense; კონტეინერს.
  4. ცადე შეცვალო minmax(150px, 1fr) 200px-ზე და ნახე როგორ გადალაგდება მოზაიკა.
