Task 4: Alignment

სწორება: justify & align

აღწერა

Grid გვაძლევს საშუალებას გავასწოროთ როგორც უჯრები კონტეინერში, ისე შიგთავსი უჯრებში.

ინსტრუქცია

  1. ამჟამად Grid არის მარცხენა ზედა კუთხეში.
  2. გააცენტრე მთლიანი Grid კონტეინერის ცენტრში:
    justify-content: center;
    align-content: center;
  3. ახლა გააცენტრე თვითონ ელემენტები თავიანთ უჯრებში:
    justify-items: center;
    align-items: center;
შენიშვნა:
  • -content სუფიქსი - მთლიან Grid-ს ეხება.
  • -items სუფიქსი - უჯრის შიგნით ელემენტს ეხება.
