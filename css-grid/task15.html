Task 15: Complex Footer

Responsive Layout with auto-fit

აღწერა

ფუტერები ხშირად შეიცავენ ბევრ სვეტს. Flexbox-ით ძნელია იმის კონტროლი, რომ ისინი თანაბრად გადანაწილდნენ და მობილურზე ავტომატურად ჩამოვიდნენ.

ინსტრუქცია

უკვე გამოყენებულია repeat(auto-fit, minmax(200px, 1fr)).

  1. შეცვალე ბრაუზერის ზომა (ან შეავიწროვე პანელი) და ნახე როგორ იქცევა ფუტერი.
  2. თუ გინდა რომ "Newsletter" ბლოკი ყოველთვის უფრო დიდი იყოს (მაგალითად 2-ჯერ დიდი), როგორ იზამدی?
    პასუხი: auto-fit-თან span რთულია. ამიტომ აქ შეგვიძლია გამოვიყენოთ კონკრეტული minmax მნიშვნელობები ან media query.
  3. დავალება: დაამატე grid-auto-flow: dense; რათა ელემენტები უფრო მჭიდროდ განლაგდნენ შევიწროვებისას თუ ზომები განსხვავებულია.
ეს არის Grid-ის "Superpower". არანაირი Media Query!
