Task 10: Intrinsic Sizing

ზომები: min-content, max-content, fit-content

აღწერა

Grid გვაძლევს საშუალებას განვსაზღვროთ სვეტის ზომა მისი შიგთავსის მიხედვით. ეს არის Intrinsic Sizing.

ინსტრუქცია

შეცვალე grid-template-columns შემდეგი მნიშვნელობებით და დააკვირდი შედეგს:

  1. max-content: პირველი სვეტი გახადე max-content. ის გაიწელება იმდენზე, რამდენიც საჭიროა რომ ტექსტი არ ჩამოვარდეს.
  2. min-content: მეორე სვეტი გახადე min-content. ის დაპატარავდება მაქსიმალურად (ყველაზე გრძელი სიტყვის ზომამდე).
  3. fit-content(300px): მესამე სვეტი. ის იქცევა როგორც auto, მაგრამ არასდროს იქნება 300px-ზე მეტი.
grid-template-columns: max-content min-content fit-content(200px);
გამოყენება: ეს იდეალურია Sidebar-ებისთვის ან ცხრილებისთვის, სადაც ზომა შინაარსზეა დამოკიდებული.
