Task 3: Grid Areas

სახელებით განლაგება: grid-template-areas

აღწერა

Grid Areas საშუალებას გვაძლევს შევქმნათ განლაგება ვიზუალური "რუკის" დახატვით CSS-ში.

ინსტრუქცია

  1. მიანიჭე შესაბამისი grid-area სახელები შვილ ელემენტებს:
    • .area-header -> grid-area: header;
    • .area-sidebar -> grid-area: sidebar;
    • .area-main -> grid-area: main;
    • .area-ads -> grid-area: ads;
    • .area-footer -> grid-area: footer;
  2. ახლა კონტეინერზე (.task3-container) განსაზღვრე რუკა: 
    grid-template-areas: 
  "header header header"
  "sidebar main ads"
  "footer footer footer";
რჩევა: ტექსტი ბრჭყალებში უნდა ემთხვეოდეს ზუსტად იმ სახელებს, რომლებიც ელემენტებს მიანიჭეთ.
