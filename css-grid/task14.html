Editorial Design & Grid Areas

აღწერა

ჟურნალის ტიპის საიტებისთვის Grid შეუცვლელია. მოდი ავაწყოთ ასიმეტრიული განლაგება.

ინსტრუქცია

  1. მიანიჭე grid-area სახელები:
    • .hero -> hero
    • .article-list -> list
    • .ad-banner -> ad
    • .quote -> quote
  2. გამოიყენე grid-template-areas რომ შექმნა სტრუქტურა: 
    "hero hero hero list"
"hero hero hero list"
"ad   ad   quote quote"
ეს განლაგება 4 სვეტს იყენებს. "hero" იკავებს 3x2 ადგილს, "list" არის ვიწრო სვეტი მარჯვნივ, ხოლო ქვედა ნაწილი ორად იყოფა.
