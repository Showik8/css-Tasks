Task 9: Dense Packing

ალგორითმი: grid-auto-flow: dense

აღწერა

ნაგულისხმევად, Grid ტოვებს ცარიელ ადგილებს (holes) თუ ელემენტი არ ეტევა დარჩენილ სივრცეში. dense ალგორითმი აიძულებს Grid-ს, რომ მოძებნოს პატარა ელემენტები და შეავსოს ეს ცარიელი ადგილები გადაადგილებით.

ინსტრუქცია

  1. დააკვირდი შედეგს. ელემენტები (2), (4) და (6) დიდია, რის გამოც ჩნდება ცარიელი ადგილები.
  2. შეცვალე grid-auto-flow კონტეინერზე: 
    grid-auto-flow: dense;
  3. ნახე როგორ "ჩაიჭედა" პატარა ელემენტები დიდებს შორის არსებულ სიცარიელეში.
გაფრთხილება: Dense ცვლის ელემენტების ვიზუალურ თანმიმდევრობას DOM-ის თანმიმდევრობის მიუხედავად. ეს შეიძლება პრობლემა იყოს Accessibility-სთვის (ეკრანის მკითხველებისთვის).
