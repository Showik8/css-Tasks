Task 13: Bento Dashboard

Masterclass: Apple-style Layout

აღწერა

"Bento Grid" არის პოპულარული დიზაინ ტრენდი (Apple, Microsoft), სადაც ვიჯეტები სხვადასხვა ზომის უჯრებს იკავებენ.

ინსტრუქცია

გვაქვს 4x4 ბადე. გამოიყენე span რომ შექმნა მოზაიკა.

  1. .w-large (Weather): უნდა იყოს 2x2.
    grid-column: span 2; grid-row: span 2;
  2. .w-wide (Music): უნდა იყოს 2x1 სიგანეში.
    grid-column: span 2;
  3. .w-tall (Activity): უნდა იყოს 1x2 სიმაღლეში.
    grid-row: span 2;
  4. .w-media (Photo): დიდი სურათი მარჯვნივ ქვემოთ 2x2.
დიზაინი: დააკვირდი როგორ ქმნის ზომების ვარიაცია ვიზუალურ რიტმს. ეს Grid-ის გარეშე ძალიან რთული იქნებოდა.
