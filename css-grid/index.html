შექმნილია თანამედროვე ვებ დიზაინისთვის

Grid Container Properties

ეს თვისებები იწერება მშობელ ელემენტზე (Container).

display

განსაზღვრავს ელემენტს როგორც grid კონტეინერს.

.container {
  display: grid; /* ან inline-grid */
}
grid-template-columns / grid-template-rows

განსაზღვრავს სვეტების და სტრიქონების ზომებს. შეგვიძლია გამოვიყენოთ px, %, fr, auto.

.container {
  grid-template-columns: 100px 1fr 2fr; /* 3 სვეტი */
  grid-template-rows: 50px auto;       /* 2 სტრიქონი */
}
100px
1fr
2fr
grid-template-areas

ქმნის ვიზუალურ სტრუქტურას სახელების გამოყენებით.

.container {
  grid-template-areas: 
    "header header header"
    "sidebar main main"
    "footer footer footer";
}
gap (row-gap, column-gap)

განსაზღვრავს დაშორებას უჯრებს შორის.

.container {
  gap: 20px; /* ორივე მხარეს */
  /* ან */
  row-gap: 10px;
  column-gap: 30px;
}
justify-items / align-items

ასწორებს შიგთავსს უჯრის შიგნით (X და Y ღერძზე).

  • start - დასაწყისში
  • end - ბოლოში
  • center - ცენტრში
  • stretch - გაწელვა (default)
.container {
  justify-items: center; /* ჰორიზონტალურად */
  align-items: center;   /* ვერტიკალურად */
  /* შემოკლებით: */
  place-items: center center;
}
justify-content / align-content

ასწორებს მთლიან grid-ს კონტეინერის მიმართ (თუ grid კონტეინერზე პატარაა).

.container {
  justify-content: space-between;
  align-content: center;
  /* შემოკლებით: */
  place-content: center space-between;
}
grid-auto-flow / grid-auto-columns / grid-auto-rows

აკონტროლებს ავტომატურად შექმნილ უჯრებს (implicit grid).

.container {
  grid-auto-flow: row; /* ან column, row dense */
  grid-auto-columns: 100px;
  grid-auto-rows: minmax(50px, auto);
}

Grid Item Properties

ეს თვისებები იწერება შვილ ელემენტებზე (Items).

grid-column / grid-row

განსაზღვრავს სად იწყება და სად მთავრდება ელემენტი (ხაზების ნომრებით).

.item {
  grid-column-start: 1;
  grid-column-end: 3;
  /* შემოკლებით: */
  grid-column: 1 / 3; /* ან 1 / span 2 */
  
  grid-row: 2 / 4;
}
grid-area

ანიჭებს სახელს ელემენტს (template-areas-სთვის) ან გამოიყენება როგორც შემოკლება (row-start / col-start / row-end / col-end).

.item {
  grid-area: header;
  /* ან კოორდინატებით: */
  grid-area: 1 / 2 / 3 / 4;
}
justify-self / align-self

ცვლის ინდივიდუალური ელემენტის განლაგებას უჯრაში (გადააწერს მშობლის justify-items-ს).

.item {
  justify-self: end;
  align-self: start;
  /* შემოკლებით: */
  place-self: start end;
}

Grid Functions (ფუნქციები)

repeat()

იმეორებს ზომებს მითითებული რაოდენობით.

grid-template-columns: repeat(3, 1fr); 
/* იგივეა რაც: 1fr 1fr 1fr */
minmax()

განსაზღვრავს მინიმალურ და მაქსიმალურ ზომას.

grid-template-columns: minmax(100px, 1fr);
/* მინიმუმ 100px, მაგრამ თუ ადგილია, გაიწელოს 1fr-მდე */
auto-fit vs auto-fill

გამოიყენება რეზპონსიული დიზაინისთვის repeat()-თან ერთად.

  • auto-fill - ქმნის იმდენ სვეტს რამდენიც ეტევა, თუნდაც ცარიელი იყოს.
  • auto-fit - იგივეა, ოღონდ ცარიელ სვეტებს აქრობს და არსებულებს წელავს.
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
