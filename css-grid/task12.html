Task 12: Nested Grids

ბუდეები: Grid შიგნით Grid

აღწერა

Grid Item თავის მხრივ შეიძლება გახდეს Grid Container. ეს აუცილებელია რთული კომპონენტების ასაწყობად.

ინსტრუქცია

  1. მარჯვენა მხარე (.main-content) არის Grid Item. მაგრამ ჩვენ გვინდა რომ მის შიგნით ელემენტებიც Grid-ით დალაგდეს.
  2. გადააქციე .main-content Grid კონტეინერად:
    display: grid;
  3. გაუწერე მას 2 სვეტი და დაშორება:
    grid-template-columns: 1fr 1fr;
    gap: 10px;
შენიშვნა: უახლესი CSS მხარს უჭერს display: subgrid-საც, რაც საშუალებას აძლევს შვილს გამოიყენოს მშობლის Grid ხაზები, მაგრამ ჯერჯერობით Nested Grid ყველაზე უსაფრთხო გზაა.
