Task 7: Responsive Cards

რეზპონსიული დიზაინი: auto-fit

აღწერა

ეს არის ყველაზე ცნობილი და გამოსადეგი Grid ტექნიკა: ბარათები, რომლებიც ავტომატურად გადადიან ახალ ხაზზე ეკრანის დაპატარავებისას, Media Query-ს გარეშე!

ინსტრუქცია

  1. ჩვენ გვინდა რომ ბარათებს ჰქონდეთ მინიმუმ 250px სიგანე.
  2. თუ ეკრანზე ადგილია, ისინი უნდა გაიწელონ (1fr).
  3. თუ ადგილი არაა, უნდა ჩამოვიდნენ ქვემოთ.
  4. შეცვალე grid-template-columns შემდეგნაირად: 
    repeat(auto-fit, minmax(250px, 1fr));
დასრულება: გილოცავ! შენ გაიარე CSS Grid-ის საბაზისო და საშუალო კურსი. ახლა შეგიძლია შექმნა რთული და მოქნილი განლაგებები.
