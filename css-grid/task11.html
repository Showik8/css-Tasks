Task 11: Named Lines

მოწინავე: ხაზების სახელები

აღწერა

ნომრების (1, 2, 3) ნაცვლად, Grid საშუალებას გვაძლევს ხაზებს დავარქვათ სახელები [] ფრჩხილებში. ეს კოდს ბევრად უფრო გასაგებს ხდის.

ინსტრუქცია

  1. განსაზღვრე სვეტები სახელებით:
    grid-template-columns: [main-start] 1fr [content-start] 2fr [content-end] 1fr [main-end];
  2. ახლა განათავსე .feature-item კონტენტის ზონაში:
    grid-column: content-start / content-end;
  3. განათავსე .full-item მთლიან სიგანეზე:
    grid-column: main-start / main-end;
უპირატესობა: თუ სვეტების რაოდენობა შეიცვლება, სახელები იგივე დარჩება, ხოლო ნომრები აირეოდა.
