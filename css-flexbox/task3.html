Justify Content

აღწერა:

justify-content ათანაბრებს ელემენტებს ძირითადი ღერძის (main axis) გასწვრივ (ჰორიზონტალურად, თუ flex-direction: row).

ინსტრუქცია:

  1. იპოვე კლასი .task3-container css ფაილში.
  2. ამჟამად არის flex-start (დასაწყისში).
  3. შეცვალე და სცადე შემდეგი მნიშვნელობები:
    • center (ცენტრში)
    • flex-end (ბოლოში)
    • space-between (თანაბარი დაშორება, კიდეებში მიკროკა)
    • space-around (თანაბარი დაშორება გარშემო)
    • space-evenly (აბსოლუტურად თანაბარი დაშორება)
  4. საბოლოოდ დატოვე space-between, რომ დავალება შესრულებულად ჩაითვალოს.
