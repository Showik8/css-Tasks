Holy Grail Layout

აღწერა:

ეს არის კლასიკური "Holy Grail" განლაგება: ჰედერი, ფუტერი, და 3 სვეტი შუაში (მარცხენა მენიუ, კონტენტი, მარჯვენა მენიუ). ფუტერი ყოველთვის ბოლოში უნდა იყოს.

ინსტრუქცია:

  1. .hg-container უკვე არის Flex (column). min-height: 400px აქვს.
  2. იპოვე .hg-main-body. ეს არის შუა ნაწილი.
  3. მიეცი მას display: flex; რათა სვეტები გვერდიგვერდ დადგეს.
  4. ასევე მიეცი flex: 1;. ეს ძალიან მნიშვნელოვანია! ეს აიძულებს შუა ნაწილს გაიზარდოს და შეავსოს მთელი თავისუფალი ადგილი ჰედერსა და ფუტერს შორის.
  5. შედეგად, ფუტერი ყოველთვის ქვემოთ იქნება მოქცეული ("Sticky Footer").
  6. (სურვილისამებრ) შეგიძლია order გამოიყენო სვეტების ადგილების შესაცვლელად.
