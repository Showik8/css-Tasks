Chat Interface

აღწერა:

ჩატის ინტერფეისი საკმაოდ რთული გასაკეთებელია Flexbox-ის გარეშე. აქ გვჭირდება ვერტიკალური განლაგება და მესიჯების მარცხნივ/მარჯვნივ გადატანა.

ინსტრუქცია:

  1. იპოვე .chat-messages.
  2. მიეცი display: flex; flex-direction: column; gap: 10px;.
  3. ახლა მესიჯები ერთმანეთის ქვემოთაა, მაგრამ ყველა მარცხენა მხარეს.
  4. იპოვე .message.sent (გაგზავნილი მესიჯი).
  5. გამოიყენე align-self: flex-end; რათა ეს მესიჯები მარჯვნივ გადავიდეს.
  6. შემოსული მესიჯები (received) დარჩება მარცხნივ (flex-start).
  7. იპოვე .chat-input-area.
  8. მიეცი მასაც display: flex; რათა ინპუტი და ღილაკი გასწორდეს.
