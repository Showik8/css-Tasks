Mosaic Gallery

აღწერა:

შევქმნათ საინტერესო გალერეა, სადაც სურათები ავტომატურად ავსებენ ადგილს, მაგრამ სხვადასხვა პროპორციებით (მოზაიკის ეფექტი).

ინსტრუქცია:

  1. იპოვე .gallery-container.
  2. მიეცი display: flex; flex-wrap: wrap;.
  3. იპოვე .gallery-item.
  4. მიეცი flex-grow: 1;. ეს აიძულებს ყველა სურათს გაიზარდოს და შეავსოს ცარიელი ადგილი.
  5. შედეგად მიიღებ მაგარ ეფექტს, სადაც ბოლო ხაზზე სურათები იწელება, ხოლო სხვა ხაზებზე სხვადასხვა ზომისაა (რადგან flex-basis სხვადასხვა აქვთ კლასებით w-1, w-2).
