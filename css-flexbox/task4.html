Align Items

აღწერა:

align-items ათანაბრებს ელემენტებს ჯვარედინი ღერძის (cross axis) გასწვრივ (ვერტიკალურად, თუ flex-direction: row).

ინსტრუქცია:

  1. იპოვე კლასი .task4-container css ფაილში.
  2. ამჟამად არის stretch (გაწელვა მთელ სიმაღლეზე, თუ სიმაღლე არ აქვს მითითებული).
  3. შეცვალე და სცადე:
    • center (ცენტრში)
    • flex-start (ზემოთ)
    • flex-end (ქვემოთ)
    • baseline (ტექსტის ხაზზე)
  4. დატოვე center.
ყურადღება მიაქციე, რომ ელემენტებს სხვადასხვა სიმაღლე აქვთ, რათა ეფექტი უკეთ დაინახო.
