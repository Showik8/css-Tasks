Flex Direction

აღწერა:

flex-direction განსაზღვრავს ძირითადი ღერძის (main axis) მიმართულებას. ნაგულისხმევად ის არის row (მარცხნიდან მარჯვნივ).

ინსტრუქცია:

  1. იპოვე კლასი .task2-container css ფაილში.
  2. შეცვალე flex-direction: row და მიუთითე column. ნახე როგორ დალაგდება ელემენტები ვერტიკალურად.
  3. შემდეგ სცადე row-reverse (შებრუნებული რიგი) და column-reverse (შებრუნებული სვეტი).
  4. საბოლოოდ დატოვე column, რომ დავალება შესრულებულად ჩაითვალოს.
