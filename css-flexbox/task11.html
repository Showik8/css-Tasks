მაგალითი: ნავიგაცია

აღწერა:

ახლა ავაწყოთ რეალური ნავიგაციის პანელი (Navbar). ეს Flexbox-ის ერთ-ერთი ყველაზე ხშირი გამოყენებაა.

ინსტრუქცია:

  1. იპოვე კლასი .nav-container.
  2. გააქტიურე Flexbox: display: flex;
  3. გაშალე ლოგო და ლინკები კიდეებზე: justify-content: space-between;
  4. გააცენტრე ვერტიკალურად: align-items: center;
  5. ახლა იპოვე .nav-links (ul ელემენტი).
  6. მასაც მიეცი display: flex; რომ ლინკები გვერდიგვერდ დადგეს.
  7. დაამატე დაშორება: gap: 20px;
