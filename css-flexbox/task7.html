Gap & Order

აღწერა:

gap ამატებს დაშორებას ელემენტებს შორის. order ცვლის ელემენტის ვიზუალურ რიგს (ნაგულისხმევი 0).

ინსტრუქცია:

  1. იპოვე კლასი .task7-container css ფაილში.
  2. დაამატე gap: 20px;.
  3. იპოვე კლასი .order-target (წითელი ყუთი).
  4. ამჟამად ის არის რიგში პირველი (HTML-ის მიხედვით).
  5. მიუთითე order: 1; ან უფრო მაღალი რიცხვი, რომ გადაიტანო ბოლოში.
  6. უარყოფითი მნიშვნელობაც (order: -1) მუშაობს!
