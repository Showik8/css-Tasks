დავალება 1

შესავალი: display: flex

აღწერა:

Flexbox არის განლაგების მოდელი, რომელიც საშუალებას გვაძლევს მარტივად განვათავსოთ ელემენტები ერთ განზომილებაში (სტრიქონში ან სვეტში).

ინსტრუქცია:

  1. იპოვე კლასი .task1-container css ფაილში.
  2. ამჟამად ელემენტები ერთმანეთის ქვემოთაა (ბლოკური განლაგება). შეცვალე display: block და გახადე display: flex.
  3. დააკვირდი როგორ დალაგდებიან ელემენტები გვერდიგვერდ ავტომატურად.
შენიშვნა: როდესაც კონტეინერს ანიჭებ display: flex-ს, მისი შვილი ელემენტები ხდებიან "flex item"-ები.
