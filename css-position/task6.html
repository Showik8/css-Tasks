მიმოხილვა

როდესაც elements ერთმანეთს ეფარება, z-index განსაზღვრავს რომელი გამოჩნდება ზემოდან.

მნიშვნელოვანია: z-index მუშაობს მხოლოდ პოზიციონირებულ (relative, absolute, fixed) ელემენტებზე.
  1. პრობლემა: Dropdown მენიუ იმალება წითელი ბარათის უკან.
  2. გადაწყვეტა: იპოვე .dropdown-menu და მიანიჭე: 
    z-index: 10;
