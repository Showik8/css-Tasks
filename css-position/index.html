Task 1: Intro

Static vs Relative

მიმოხილვა

CSS-ში ყველა ელემენტი თავდაპირველად არის position: static. ეს ნიშნავს, რომ ისინი თანმიმდევრობით ლაგდებიან.

position: relative საშუალებას გვაძლევს გადავაადგილოთ ელემენტი თავდაპირველი ადგილიდან ისე, რომ სხვა ელემენტებს ხელი არ შეეშალოთ.

  1. პოზიციონირების შეცვლა: იპოვე .box-2 სტილების ფაილში.
  2. მიანიჭე: 
    position: relative;
  3. გადაწიე ის თავდაპირველი ადგილიდან: 
    top: -20px;
left: 20px;
დააკვირდი: ყუთის 'ადგილი' ცარიელი რჩება, სხვა ყუთები არ ინაცვლებენ. ეს არის relative-ის მთავარი თვისება.
