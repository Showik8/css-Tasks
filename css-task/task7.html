ფორმები (Inputs)

ინსტრუქცია:

ველები, სადაც მომხმარებელს ტექსტი შეჰყავს, ლამაზი უნდა იყოს. იპოვე .styled-input.

  1. შეცვალე ჩარჩოს ფერი: border: 2px solid #eee;
  2. როდესაც მომხმარებელი დააკლიკებს (focus), ჩარჩო გახდეს ლურჯი. ამისთვის დაწერე ახალი წესი: 
    .styled-input:focus {
    outline: none;
    border-color: #61dafb;
}
  3. დაამატე ქვედა ჩრდილი: box-shadow: inset 0 1px 3px rgba(0,0,0,0.1);
