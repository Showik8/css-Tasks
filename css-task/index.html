შესავალი CSS-ში

ინსტრუქცია:

ამ დავალებაში შენ უნდა შეცვალო მარჯვენა მხარეს არსებული ელემენტების სტილი style.css ფაილის გამოყენებით.

  1. იპოვე კლასი .box css ფაილში და შეცვალე მისი ფერი (background-color) წითლად (red).
  2. გაუზარდე ზომები ყუთს: სიგანე (width) გახადე 200px.
  3. ტექსტი გაათეთრე (color: white) და გა ცენტრე (text-align: center).
  4. დაამრგვალე კუთხეები 10px-ით (border-radius).
რჩევა: ცვლილებების სანახავად დაარეფრეშე გვერდი ბრაუზერში.
