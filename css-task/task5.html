ინტერაქცია (Hover & Transition)

ინსტრუქცია:

მოდი ღილაკი გავაცოცხლოთ. როდესაც მაუსს მიიტან, მან ფერი უნდა შეიცვალოს.

იპოვე სექცია "დავალება 5".

  1. შექმენი ახალი წესი: 
    .btn:hover {
    background-color: #61dafb;
    color: #1e1e24;
    transform: translateY(-5px);
}
  2. დააკვირდი transition: all 0.3s;-ს მთავარ .btn კლასში. სცადე მისი წაშლა და ნახე განსხვავება, შემდეგ დააბრუნე.
