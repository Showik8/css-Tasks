Task 2: Focus

Accessibility & Keyboard States

მიმოხილვა

:focus არის კრიტიკული ხელმისაწვდომობისთვის (Accessibility). ის გვაჩვენებს სად არის კურსორი კლავიატურის გამოყენებისას.

  1. ინპუტის ფოკუსი: იპოვე .accessible-input და გაუკეთე ლამაზი ფოკუსი: 
    .accessible-input:focus {
    border-color: #3b82f6; 
    box-shadow: 0 0 0 4px rgba(59, 130, 246, 0.2);
    background-color: white;
}
  2. მშობლის ფოკუსი (:focus-within): ეს ფსევდო-კლასი აქტიურდება, როცა შვილ ელემენტს აქვს ფოკუსი. იპოვე .focus-group: 
    .focus-group:focus-within {
    border-color: #22c55e;
    background-color: #f0fdf4;
    border-style: solid;
}
შენიშვნა: ყოველთვის გამოიყენე outline ან box-shadow ფოკუსისთვის. არასდროს გათიშო ის (outline: none) ალტერნატივის გარეშე!
