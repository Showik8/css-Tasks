Custom Toggles & Radios

მიმოხილვა

:checked ფსევდო-კლასი გვაძლევს საშუალებას შევქმნათ სრულიად ქასთომ Checkbox-ები და Radio ღილაკები.

  1. Toggle Switch: ჩვენ დავმალეთ სტანდარტული checkbox. ახლა უნდა შევცვალოთ .custom-toggle, როცა input მონიშნულია. გამოვიყენოთ + (adjacent sibling) სელექტორი: 
    /* როცა ჩეკბოქსი მონიშნულია, შეცვალე მისი მეზობელი .custom-toggle */
input:checked + .custom-toggle {
    background-color: #22c55e;
}

/* გადავაადგილოთ წრე */
input:checked + .custom-toggle::after {
    transform: translateX(22px);
}
