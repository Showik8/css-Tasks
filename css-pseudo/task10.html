Task 10: Logic

Has, Not, Is, Where

მიმოხილვა

თანამედროვე CSS გვაძლევს ლოგიკურ ოპერატორებს.

  1. :has() - მშობლის არჩევა: აირჩიე ბარათი, რომელსაც აქვს .pro კლასის მქონე შვილი. 
    .logic-card:has(.pro) {
    border-color: #0ea5e9;
    background-color: #f0f9ff;
    box-shadow: 0 4px 6px rgba(14, 165, 233, 0.1);
}
  2. :not() - გამორიცხვა: აირჩიე ყველა, გარდა .pro ბარათისა. 
    .logic-card:not(:has(.pro)) {
    opacity: 0.7;
}
  3. :is() - დაჯგუფება: 
    .logic-card :is(.badge, .icon-box) {
    border-radius: 50%; /* ორივეს ერთად */
}
:has() არის რევოლუცია CSS-ში!
