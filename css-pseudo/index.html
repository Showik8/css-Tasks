Task 1: Intro

Hover & Active Interactions

მიმოხილვა

ფსევდო-კლასები (Pseudo-classes) გვეხმარება ელემენტის სტილი შევცვალოთ მისი მდგომარეობის მიხედვით.

ყველაზე ხშირად გამოყენებადი მდგომარეობებია მაუსის გადატარება (Hover) და დაჭერა (Active).

  1. ღილაკის ანიმაცია: იპოვე .magic-button და დაამატე: 
    .magic-button:hover {
    background-color: #2563eb;
    transform: translateY(-2px);
    box-shadow: 0 10px 15px -3px rgba(37, 99, 235, 0.4);
}

.magic-button:active {
    transform: translateY(0);
    box-shadow: 0 4px 6px -1px rgba(37, 99, 235, 0.2);
}
  2. ბარათის ეფექტი: იპოვე .card-hover და დაამატე: 
    .card-hover:hover {
    border-color: #38bdf8;
    background-color: #f0f9ff;
    transform: scale(1.05) rotate(1deg);
    color: #0284c7;
}
რჩევა: transition თვისება უკვე დამატებულია ძირითად კლასებზე, ამიტომ ცვლილება ავტომატურად "ანიმირებული" იქნება.
