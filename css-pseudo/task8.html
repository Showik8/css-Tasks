Task 8: Structure

First, Last & Only Child

მიმოხილვა

ეს სელექტორები გვეხმარება ავირჩიოთ ელემენტები მათი განლაგების მიხედვით მშობელში.

  1. პირველი და ბოლო: 
    .pill-item:first-child {
    background-color: #3b82f6; 
    color: white;
}
.pill-item:last-child {
    background-color: #ef4444; 
    color: white;
}
  2. ერთადერთი შვილი (:only-child): ეს მუშაობს, თუ ელემენტი მარტოა მშობელში. 
    .pill-item:only-child {
    width: 100%;
    text-align: center;
    background-color: #8b5cf6;
    color: white;
}
