Task 5: Form States

Enabled, Disabled & Placeholders

მიმოხილვა

ფორმის ელემენტები რეაგირებენ თავიანთ სტატუსზე.

  1. Enabled / Disabled: გათიშული ელემენტები უნდა გამოიყურებოდეს არააქტიურად. იპოვე .state-input:disabled: 
    .state-input:disabled {
    background-color: #f1f5f9;
    color: #94a3b8;
    cursor: not-allowed;
    border-color: #e2e8f0;
}
.state-input:enabled {
    background-color: #fff;
    border-color: #cbd5e1;
}
  2. Placeholder-shown: ეს ფსევდო-კლასი აქტიურია, როცა ინპუტი ცარიელია (და placeholder ჩანს). იპოვე .state-input:placeholder-shown: 
    /* როცა ცარიელია */
.state-input:placeholder-shown {
    border-style: dashed;
}

/* როცა შევავსებთ (placeholder გაქრება) */
.state-input:not(:placeholder-shown) {
    border-style: solid;
    border-color: #22c55e;
    background-color: #f0fdf4;
}
