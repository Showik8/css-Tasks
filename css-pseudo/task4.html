მიმოხილვა

ბმულებს (Links) აქვთ 4 ძირითადი მდგომარეობა: :link, :visited, :hover, :active.

უსაფრთხოების გამო, :visited-ზე შეგვიძლია შევცვალოთ მხოლოდ ფერი.

  1. ვიზიტორთა ისტორია: იპოვე .history-link: 
    /* ნანახი ლინკები */
.history-link:visited {
    color: #9333ea; /* Purple */
    text-decoration: line-through;
}

/* არანახი ლინკები */
.history-link:link {
    color: #2563eb; /* Blue */
}
