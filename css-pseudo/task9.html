Nth-Child & Counting

მიმოხილვა

:nth-child(formula) ყველაზე ძლიერი სტრუქტურული სელექტორია.

  1. Odd / Even: 
    .grid-box:nth-child(odd) {
    background-color: #cbd5e1;
}
  2. ფორმულა (an+b): მაგალითად, ყოველი მეოთხე ელემენტი. 
    .grid-box:nth-child(4n) {
    background-color: #f59e0b;
    color: white;
}
  3. პირველი 3 ელემენტი: 
    .grid-box:nth-child(-n+3) {
    border-bottom: 4px solid #3b82f6;
}
